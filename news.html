News Samba

Stay up to date with the latest developments from the Samba project. We continue working to improve compatibility, performance, and security across all supported platforms.

Our most recent updates include enhancements to Active Directory integration, optimizations in file-sharing performance, and several important security patches. The development team is also preparing new features that will be introduced in the upcoming release cycle.

Check this space regularly for announcements, release notes, and relevant community updates as we advance the evolution of Samba.
